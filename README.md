## tamo-cli

`tamo-cli` is a command-line utility that displays TAMO homework and grades data.

---

## Installation

Clone the repository and run:

```bash
sudo ./install.sh
```

If it doesn’t run:

```bash
chmod +x install.sh
```

---

## Usage

```bash
tamo
tamo grades
tamo homework
```
